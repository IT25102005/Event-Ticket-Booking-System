API Endpoints

Base URL: http://localhost:8080/api


Payments:

  • POST /payments - Create new payment
  • GET /payments - Get all payments
  • GET /payments/{id} - Get payment by ID
  • PUT /payments/{id}/status?status=COMPLETED - Update status
  • DELETE /payments/{id} - Delete payment

Refunds:

  • POST /refunds - Create refund request
  • GET /refunds - Get all refunds
  • GET /refunds/{id} - Get refund by ID
  • PUT /refunds/{id}/approve?approvedBy=Admin - Approve refund
  • PUT /refunds/{id}/reject - Reject refund
  • DELETE /refunds/{id} - Delete refund
